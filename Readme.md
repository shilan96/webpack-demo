安装依赖的方法: 区分生产环境和开发环境, 不建议安装到全局, 可能会造成版本的错误
	npm install  ----安装全部的依赖包
	npm install --production  ----只安装生产环境的依赖包
	npm install [name]  ----安装到全局
	npm install --save [name]  ----安装到生产环境
	npm install --save-dev [name]  ----安装到开发环境
	npm install -D [name]  ----上一条的简写
开发环境
	devDependencies 开发依赖
生产环境
	dependencies 生产依赖
